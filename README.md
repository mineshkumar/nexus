# Nexus Cloud Launchpad

A modern, glassmorphism-styled dashboard for managing your most used web applications and shortcuts. Built with performance and aesthetics in mind.

## Features

- **Cloud Sync**: instantly syncs your shortcuts across devices using Supabase.
- **Drag & Drop**: reorganize your launchpad grid with an intuitive drag-and-drop interface.
- **Glassmorphism UI**: a premium, translucent design using modern CSS backdrop-filters.
- **Easy Management**: add, edit, and delete shortcuts directly from the dashboard.

## Tech Stack

- **HTML5 & Vanilla JS**: Lightweight core with no heavy framework overhead.
- **Tailwind CSS**: Utility-first styling for rapid UI development.
- **Supabase**: Backend-as-a-Service for real-time database capabilities.
- **Lucide Icons**: Beautiful, consistent open-source icons.
- **SortableJS**: Smooth drag-and-drop functionality.

## Setup

1. Clone the repository.
2. Open `index.html` in your browser.
3. Ensure you have your Supabase credentials configured in the script section of `index.html`.
